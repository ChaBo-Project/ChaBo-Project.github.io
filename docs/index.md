# ChaBo Microservices

ChaBo is an open chatbot ecosystem built around reusable services, shared
documentation, and implementation examples.

The goal of this repository is to make new chatbot implementations easier to
build, operate, and contribute back to the ecosystem.

## Website Relationship

The existing ChaBo website remains available as the public landing page:

[https://chabo-project.github.io/chabo/](https://chabo-project.github.io/chabo/)

This site is the technical documentation space for reusable microservices,
integration guidance, deployment notes, and contributor-facing architecture.

## What Belongs Here

- Reusable microservices for chatbot implementations.
- Documentation for each service.
- Example integrations and deployment patterns.
- Contribution guidelines for implementers.
- Architecture decisions that help teams align on standards.

## Current Work Areas

- WhatsApp channel microservice.
- Speech channel microservice.
- Management backend specification.
- Documentation website and microservice documentation standard.
- UAT deployment and demonstration environment.
